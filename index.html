HTML

  • The "head" element contains information about the webpage that is used by the browser and is not readable to the user on the webpage
  • The "body" element contains the visible content that is shown to the user.
  • The "header" element contains only the information shown on the header portion of the webpage

CSS

  • A "." preceding a class designates a class selector (was used with .card)
  • Using <link rel="stylesheet" href="./assets/style.css"> in the head of your file, you can add your css elements to your webpage by referencing the stylesheet file.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Multiple classes can be changed by adding a comma after each class before starting your changes with {

GIT

  • The steps to commit a file to the repo are as follows:
    1. git status --Start by checking the status of your current branch
    2. (if not in specified branch) git checkout -b (your new branch here) --This will allow your new changes to be committed to a seperate branch of your Github repo
    3. git add -A --This cashes your changes in a staging area to be committed
    4. git commit -m "your comment here" -- This commits your changes to your working branch
    5. git pull origin main --This pulls any changes from the main branch to your working branch to avoid conflcts
    6. git push origin (your branch name here) --this pushes your changes to the working branch of the repo and allows your new changes to be reviewed
    7. After your changes have been pushed, then in Github the changes can be pulled into the main code
  • git status --checks what branch you are in
  • git checkout -b (branch name) creates a new branch and switches to it

JAVASCRIPT

  • You can add a javascript file to your HTML file with <script src="your .js file path here"></script> in the footer of your HTML file
  • A variable (var) is a named container that allows us to store data in our code.
  • Control flow is the order in wheich a computer executes code in our script. In javascript, code is ran from the first statement to the last in order. This can be changed with conditional statements and loops.
  • Conditional statements are as follows: if, else, else if, switch.
  • An array is a single variable that holds a group of data. They are indexed in 0,1,2,3,4,... and can hold any type of data (string, numeric, boolean)
  • Using the (var).length array property we can identify the variable in the array in our code.
  • A function is a block of code that can be called in another part of your code.
