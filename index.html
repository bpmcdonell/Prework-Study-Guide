HTML

  • The "head" element contains information about the webpage that is used by the browser and is not readable to the user on the webpage
  • The "body" element contains the visible content that is shown to the user.
  • The "header" element contains only the information shown on the header portion of the webpage

GIT

  • The steps to commit a file to the repo are as follows:
    1. "git status" --Start by checking the status of your current branch
    2. (if not in specified branch) "git checkout -b (!--your new branch here--)" --This will allow your new changes to be committed to a seperate branch of your Github repo
    3. "git add -A" --This cashes your changes in a staging area to be committed
    4. "git commit -m "your comment here"" -- This commits your changes to your working branch
    5. "git pull origin main" --This pulls any changes from the main branch to your working branch to avoid conflcts
    6. "git push origin (your branch name here) --this pushes your changes to the working branch of the repo and allows your new changes to be reviewed
    7. After your changes have been pushed, then in Github the changes can be pulled into the main code
